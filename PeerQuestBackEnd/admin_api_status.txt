# API endpoints for admin panel are now available:
#
# - GET    /api/users/admin/users/           (list/search users, flags inappropriate usernames)
# - POST   /api/users/admin/users/<uuid>/ban/    (ban user, with reason and optional expiration)
# - POST   /api/users/admin/users/<uuid>/unban/  (unban user)
# - DELETE /api/users/admin/users/<uuid>/delete/ (delete user)
#
# Only superusers can access these endpoints.
#
